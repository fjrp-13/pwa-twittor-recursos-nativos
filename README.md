# Twittor:

Un cascarón de chat usando jQuery para PWAs con uso de recursos nativos:
* cámara
* ubicación gps

# Notas:
Recuerden que deben de reconstruir los módulos de node con el comando

```
npm install
```

Luego, para correr en producción
```
npm start
```

Para correr en desarrollo
```
npm run dev
```
